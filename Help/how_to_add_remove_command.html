How to add Command

Rules to follow when adding a command

  • When adding a command you have to choose a command in accordance to the satellite functionalities
  • An added command have to be nammed in a meaningful way corresponding to its functionality
  • The choosen initials for satellite command can't exceed four letters in maximum
  • The command names have to be clear and concise

Steps in adding command

  1. Go to the settings menu
  2. From settings go to add mission command submenu
  3. From add mission command go to new command to add
  4. Then add the full command name
  5. Then add command initials
  6. Then click on add to add your sattelite mission command

Steps in removing command

  1. Go to the settings menu
  2. From settings go to Remove mission command submenu
  3. From Remove mission command go to command to remove
  4. Then add the full command name for the command you want to remove
  5. Then add the command initials
  6. Then click on remove to remove your sattelite mission command you don't want
